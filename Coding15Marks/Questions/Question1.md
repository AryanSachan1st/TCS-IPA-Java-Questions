# Question 1 — Count Lowercase Letters

Given the string `"DataBaSe"`, write a Java program to count and print the number of **lowercase letters** present in the string.

## Example

**Input (hardcoded):**
```
str = "DataBaSe"
```

**Output:**
```
4
```

## Constraints
- Do **not** use any built-in `Character.isLowerCase()` method; compare characters directly using ASCII range `'a'` to `'z'`.
- Iterate through the string character by character.
