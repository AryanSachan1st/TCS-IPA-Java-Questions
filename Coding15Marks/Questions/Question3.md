# Question 3 — Count Spaces and Non-Space Characters

Given the sentence `"Hello there Welcome back to my channel"`, write a Java program to count and print:
1. The **number of spaces** in the sentence.
2. The **number of non-space characters** in the sentence.

## Example

**Input (hardcoded):**
```
sentence = "Hello there Welcome back to my channel"
```

**Output:**
```
Num of Spaces: 6
Num of Characters: 32
```

## Constraints
- Iterate through the sentence character by character.
- Count spaces by checking if each character equals `' '`.
- Derive the character count as: `total length − number of spaces`.
